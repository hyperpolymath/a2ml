= A2ML Ada/SPARK TUI (Prototype)

This is a minimal Ada TUI wrapper around the A2ML CLI.

== Notes

* Uses `GNAT.OS_Lib.Spawn` to run `just cli ...` commands.
* Marked `SPARK_Mode (Off)` because process spawning is not SPARK-compatible.

== Build

[source,bash]
----
cd prototype/ada-tui

gnatmake -gnat2022 -O2 -o a2ml-tui src/main.adb
./a2ml-tui
----

== Requirements

* GNAT / Ada compiler
* `just` available in PATH
* A2ML CLI prototype (ReScript) buildable on this machine
